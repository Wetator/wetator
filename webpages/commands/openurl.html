Open Url

Description

Open the specified url.

First Parameter

Required
The url to open.

Use a relative url here to be independet of the server the application is deployed on.
The server url is defined in the configuration.

Second Parameter (not used)

-

Examples

Config

...
web.baseUrl=http://wet.rbri.org
...

Open url 'http://wet.rbri.org/commands/openurl.html'
